About ALUSOURCE

Built for ALU Researchers

ALUSOURCE is a student-built digital research library developed as part of the ALU Building Responsive UI capstone module. It connects students across the BSE, BEL, and IBT Final Year programs with facilitator-curated academic resources — enabling structured reading, note-taking, and progress tracking.

The platform demonstrates semantic HTML, mobile-first CSS, vanilla JavaScript (IIFE modules), regex-powered search and validation, localStorage persistence, and WCAG AA accessibility compliance.
