About

📚

ALU Library

Capstone Reading Tracker · Final Year 2025/26

What is this?

A personal reading tracker built specifically for ALU final-year students managing their capstone research. Track every book you read, are reading, or plan to read. Search with regex, sort by any field, and export your library as JSON.

Features

  • 📋 Track books across 3 statuses — Want to Read, Reading, Finished
  • 🔍 Live regex search with match highlighting
  • 📊 Dashboard with stats, progress bar, 7-day chart
  • 🎯 Set a capstone reading goal and track progress
  • ✏️ Inline edit and delete with confirmation
  • 📦 Import/export JSON for backup
  • 🌙 Dark/light mode toggle
  • ⌨️ Full keyboard navigation
  • 📱 Responsive — works on phone, tablet, desktop

Regex Patterns You Can Use

  • /^The/i — books starting with "The"
  • /(design|product)/i — design OR product books
  • /\b(\w+)\s+\1\b/i — duplicate words in title
  • /^[A-Z][a-z]+\s[A-Z]/ — "Firstname Lastname" author format
